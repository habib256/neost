Aide & raccourcis
  • Souris : cliquez dans l'écran pour la capturer (curseur GEM), Suppr pour la libérer. Échap est envoyée au ST.
  • Clavier : routé vers l'IKBD (disposition US). Joystick clavier : flèches + Ctrl droit = bouton feu (port 1).
  • Disquettes de jeu : « Monter A: + Reset » bascule au besoin sur un vrai TOS (EmuTOS ne boote que les disquettes système) — TOS 2.06 sur STE/Mega STE, TOS 1.0x sur ST. Choisissez une ROM à la main pour figer ce choix.
  • Glisser-déposer : lâchez une image .st / .msa / .dim / .stx directement sur l'écran pour la monter en A: et rebooter.
  • Machine & RAM : figées à la construction de la machine → leur changement recharge la page (paramètres d'URL ?machine= et ?mem=, partageables).
  • Son : le navigateur exige un geste utilisateur — cliquez une fois dans l'écran pour débloquer l'audio (YM2149 + DMA STE + bruits de lecteur).
